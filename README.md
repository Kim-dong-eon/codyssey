최소한의 예시로 학습: AI 모델에게 방대한 데이터를 새로 학습시키는 대신, 단 몇 개(Few)의 예시만 보여주고 새로운 작업을 수행하게 하는 기법입니다.

신속한 적응력: 이미 대량의 지식을 배운 거대 언어 모델(LLM)이 추가적인 미세조정(Fine-tuning) 없이도 새로운 규칙이나 문맥을 빠르게 이해하도록 돕습니다.

프롬프트 활용: 사용자가 프롬프트(명령어)에 "질문-답변" 같은 모범 사례를 입력해 주는 것만으로 AI의 답변 정확도와 일관성을 크게 높일 수 있습니다.
1. 감정 분석 (Sentiment Analysis)
[프롬프트 입력 예시]
다음 문장의 감정을 [긍정], [부정], [중립] 중 하나로 분류해줘.

예시 1: "배송도 빠르고 상품 상태도 아주 마음에 듭니다." -> [긍정]
예시 2: "가격은 싼데 내구성이 너무 떨어져서 금방 망가졌어요." -> [부정]
예시 3: "오후 3시에 택배가 도착했습니다." -> [중립]

실제 문제: "디자인은 예쁜데 사이즈가 생각보다 조금 작네요." ->

AI의 예상 답변: [부정] (또는 맥락에 따라 [중립])

2. 일관된 형식의 데이터 변환
[프롬프트 입력 예시]
텍스트에서 이름과 전화번호만 뽑아서 JSON 형식으로 정리해줘.

예시 1: "서울에 사는 김철수 씨는 010-1234-5678입니다." -> {"name": "김철수", "phone": "010-1234-5678"}
예시 2: "담당자 이영희 대리 연락처: 010-9876-5432" -> {"name": "이영희", "phone": "010-9876-5432"}

실제 문제: "문의사항은 박민수 팀장(010-5555-4444)에게 연락 바랍니둥." ->

AI의 예상 답변: {"name": "박민수", "phone": "010-5555-4444"}

3. 특정 사투리나 말투 변환
[프롬프트 입력 예시]
표준어 문장을 친근한 경상도 사투리로 바꿔줘.

예시 1: "오늘 점심은 무엇을 먹을 예정인가요?" -> "오늘 점심 뭐 먹을낀데?"
예시 2: "이 물건은 도대체 어디에 쓰는 물건이지?" -> "이거는 대체 어따 쓰는 기고?"

실제 문제: "방금 한 말은 정말 신기하고 대단하네요." ->

AI의 예상 답변: "방금 한 말 진짜 신기하고 직이네!" (또는 "방금 한 말 식겁하겠네예!")
